Pre-Arrival Setup (IT Staff ‚Äî Complete Before Start Date)

1
Create user account (Google Workspace)

Create account following naming convention: firstname.lastname@company.com. Add to appropriate organizational unit (OU) based on department and role.

  • Set temporary password (require change on first login)
  • Assign to correct department group
  • Enable appropriate Google services (Gmail, Drive, Calendar, Meet)
2
Provision equipment in Snipe-IT

Check out the following from inventory:

  • Laptop (standard: Dell Latitude 5540 or equivalent)
  • Monitor (if office-based: Dell 24" P2422H)
  • Keyboard & mouse
  • Docking station (if applicable)
  • Headset (for remote/clinical staff)

In Snipe-IT: Check out all assets to the new user. Record asset tags, serial numbers, and condition.

3
Configure laptop
  • Image with standard SOE (Standard Operating Environment)
  • Install required software: Chrome, Office 365, VPN client, ScreenConnect agent
  • Apply security policies (BitLocker encryption, firewall, antivirus)
  • Join to domain (if applicable)
  • Install EHR client (clinical staff only ‚Äî IntakeQ or applicable system)
4
Configure application access

Based on role and department, grant access to:

  • All staff: Email, Drive, Calendar, GLPI (self-service portal)
  • Clinical staff: EHR (IntakeQ), clinical documentation systems
  • Administrative: Billing systems, scheduling software
  • IT staff: Admin consoles, ScreenConnect, Snipe-IT
5
Create GLPI ticket for onboarding

Create a ticket (Category: Onboarding ‚Üí New Hire Setup) tracking all setup tasks. Attach the completed checklist when done.

6
Schedule Day 1 IT orientation

Block 30-45 minutes on the new hire's calendar for IT orientation covering equipment handoff, account activation, security training enrollment, and Q&A.

Day 1: IT Orientation (New Employee)

üëã Welcome! Here's what to expect during your IT setup on Day 1. Your IT contact will walk you through each step.
1
Equipment handoff

You'll receive your laptop, monitor, and accessories. IT will verify everything powers on and walk you through basic setup. Please sign the equipment acknowledgment form.

2
Account activation

Log in to your Google Workspace account using the temporary credentials provided by IT. You'll be prompted to:

  • Set a new password (see password policy)
  • Configure Multi-Factor Authentication (MFA) using your phone
  • Set up password recovery options
3
Security training enrollment

You'll be enrolled in KnowBe4 security awareness training. Complete the following within your first week:

  • Security Awareness Fundamentals (20 min)
  • HIPAA for Healthcare Workers (30 min) ‚Äî Required for all staff
  • Phishing Identification (15 min)
4
VPN setup (remote workers)

If you'll work remotely, IT will install and configure the VPN client on your laptop. See VPN Troubleshooting for future reference.

5
Application walkthrough

IT will verify you can access all required applications and briefly demonstrate:

  • How to submit IT support tickets via GLPI
  • How to use the self-service password reset portal
  • How to report phishing emails (see Phishing Reporting Guide)
